Triết lý thiết kế & Tiêu chuẩn phát triển

YAGNI (You Aren't Gonna Need It)

Loại bỏ gánh nặng runtime. Biên dịch tĩnh 1 lần duy nhất offline, triển khai tĩnh CDN chi phí 0đ, không upload API phức tạp.

DRY (Don't Repeat Yourself)

Lưu trữ index-mapped JSON phẳng cho văn bản dịch. Kế thừa 100% tọa độ CSS từ HTML absolute gốc, không sao chép style.

Trải nghiệm tối cao (UX-first)

Tự động dồn dòng chống đè chữ (Layout Shift Engine), tối ưu hóa cảm ứng chạm mượt mà trên iOS/Android và phím tắt Desktop.
