agent LangGraph tự động tạo các truy vấn tìm kiếm sử dụng các mô hình Google Gemini
và tích hợp nghiên cứu web qua Google Search API. Hệ thống sử dụng
lập luận phản chiếu để xác định các khoảng trống kiến thức, tinh chỉnh tìm kiếm lặp đi lặp lại, và
tổng hợp câu trả lời với trích dẫn. Frontend và backend hỗ trợ hot-reloading.
Cấu trúc dự án bao gồm các thư mục frontend/ và backend/ riêng biệt.
Yêu cầu thiết lập bao gồm Node.js, npm, Python 3.8+, và khóa Google Gemini API.
Sau khi cấu hình khóa API trong file .env của backend, các phụ thuộc cho cả
backend (sử dụng pip install .) và frontend (npm install) có thể được cài đặt. Các máy chủ phát triển
có thể chạy đồng thời với make dev hoặc riêng lẻ. Agent backend,
được định nghĩa trong backend/src/agent/graph.py, tạo các truy vấn tìm kiếm ban đầu, tiến hành nghiên cứu web,
thực hiện phân tích khoảng trống kiến thức, tinh chỉnh truy vấn lặp đi lặp lại, và
tổng hợp câu trả lời có trích dẫn sử dụng model Gemini. Triển khai sản xuất liên quan đến
máy chủ backend cung cấp bản dựng frontend tĩnh và yêu cầu Redis để streaming
đầu ra thời gian thực và cơ sở dữ liệu Postgres để quản lý dữ liệu. Hình ảnh Docker có thể được
xây dựng và chạy sử dụng docker-compose up, cũng yêu cầu LangSmith API key cho
ví dụ docker-compose.yml. Ứng dụng sử dụng React với Vite, Tailwind
CSS, Shadcn UI, LangGraph, và Google Gemini. Dự án được cấp phép theo
Apache License 2.0.
# Tạo Agent Graph của chúng ta
builder = StateGraph(OverallState, config_schema=Configuration)
# Định nghĩa các nút chúng ta sẽ chu kỳ giữa
builder.add_node("generate_query", generate_query)
builder.add_node("web_research", web_research)
builder.add_node("reflection", reflection)
builder.add_node("finalize_answer", finalize_answer)
# Đặt điểm vào là `generate_query`
# Điều này có nghĩa là nút này là nút đầu tiên được gọi
builder.add_edge(START, "generate_query")
# Thêm cạnh có điều kiện để tiếp tục với các truy vấn tìm kiếm trong một nhánh song song
builder.add_conditional_edges(
"generate_query", continue_to_web_research, ["web_research"]
)
# Phản chiếu về nghiên cứu web
builder.add_edge("web_research", "reflection")
# Đánh giá nghiên cứu
builder.add_conditional_edges(
"reflection", evaluate_research, ["web_research",
"finalize_answer"]
)
