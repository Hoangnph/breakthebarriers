Script Python này định nghĩa một function duy nhất tên greet, nhận tên của một người
và trả về một lời chào được cá nhân hóa. Decorator @tool() phía trên function này
tự động đăng ký nó như một tool mà AI hoặc chương trình khác có thể dùng. Documentation string
và type hint của function được FastMCP dùng để cho Agent biết
tool hoạt động như thế nào, cần input gì, và sẽ trả về gì.
Khi script được thực thi, nó khởi động FastMCP server, lắng nghe request
trên localhost:8000. Điều này làm cho function greet khả dụng như một network service. Một
agent sau đó có thể được cấu hình để kết nối tới server này và dùng tool greet để
tạo lời chào như một phần của tác vụ lớn hơn. Server chạy liên tục cho tới khi
được dừng thủ công.
Sử dụng FastMCP Server với ADK Agent
Một ADK agent có thể được thiết lập như một MCP client để dùng FastMCP server đang chạy. Việc này
đòi hỏi cấu hình HttpServerParameters với địa chỉ mạng của FastMCP server,
thường là http://localhost:8000.
Có thể thêm tham số tool_filter để giới hạn việc dùng tool của agent vào các
tool cụ thể do server cung cấp, chẳng hạn 'greet'. Khi được prompt với request như "Greet
John Doe," LLM nhúng trong agent nhận diện tool 'greet' có sẵn qua MCP,
gọi nó với argument "John Doe," và trả về response của server. Quy trình này
minh họa việc tích hợp các tool do người dùng định nghĩa và expose qua MCP
với một ADK agent.
