Hình 2: Mẫu thiết kế Tool use
Các điểm chính
Tool Use (Function Calling) cho phép agent tương tác với các hệ thống bên ngoài và
truy cập thông tin động.
Nó bao gồm việc định nghĩa tool với mô tả và tham số rõ ràng để LLM
có thể hiểu.
LLM quyết định khi nào dùng tool và tạo các function call có cấu trúc.
Các framework agentic thực thi các lời gọi tool thực tế và trả kết quả về cho
LLM.
Tool Use là thiết yếu để xây dựng các agent có thể thực hiện hành động trong thế giới thực
và cung cấp thông tin cập nhật.
LangChain đơn giản hóa việc định nghĩa tool bằng decorator @tool và cung cấp
create_tool_calling_agent cùng AgentExecutor để xây dựng các agent dùng tool.
