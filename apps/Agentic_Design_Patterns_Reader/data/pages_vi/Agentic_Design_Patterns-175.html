`npx` (Node Package Execute), đi kèm với npm (Node Package Manager) phiên bản
5.2.0 trở lên, là một utility cho phép thực thi trực tiếp các package Node.js từ
npm registry. Điều này loại bỏ nhu cầu cài đặt global. Về bản chất, `npx` đóng vai trò
như một npm package runner, và thường được dùng để chạy nhiều MCP server cộng đồng
được phân phối dưới dạng package Node.js.
Việc tạo file __init__.py là cần thiết để đảm bảo file agent.py được nhận diện là
một phần của Python package có thể discovery bởi Agent Development Kit (ADK). File này
nên nằm trong cùng thư mục với agent.py.
# ./adk_agent_samples/mcp_agent/__init__.py
