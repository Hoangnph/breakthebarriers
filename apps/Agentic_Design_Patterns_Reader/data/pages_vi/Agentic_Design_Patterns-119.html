Code thực hành (Crew AI)
Đoạn code Python này định nghĩa một crew vận hành bằng AI dùng framework CrewAI để
tạo một bài blog về xu hướng AI. Nó bắt đầu bằng cách thiết lập môi trường, tải
API key từ file .env. Phần lõi của ứng dụng bao gồm định nghĩa hai agent: một
researcher để tìm và tóm tắt xu hướng AI, và một writer để tạo bài blog dựa
trên nghiên cứu.
Hai task được định nghĩa tương ứng: một task để nghiên cứu xu hướng và task khác để
viết bài blog, trong đó task viết phụ thuộc vào đầu ra của task nghiên cứu
. Các agent và task này sau đó được lắp ghép thành một Crew, chỉ định một quy trình tuần tự
trong đó các task được thực thi theo thứ tự. Crew được khởi tạo với các agent,
task và một mô hình ngôn ngữ (cụ thể là model "gemini-2.0-flash"). Hàm main
thực thi crew này bằng phương thức kickoff(), điều phối
sự cộng tác giữa các agent để tạo đầu ra mong muốn. Cuối cùng, code
in kết quả cuối cùng của quá trình thực thi crew, tức bài blog được tạo.
