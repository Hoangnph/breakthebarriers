ADK agent và dùng HttpServerParameters để thiết lập kết nối với
FastMCP server đang hoạt động.
# ./adk_agent_samples/fastmcp_client_agent/agent.py
import os
from google.adk.agents import LlmAgent
from google.adk.tools.mcp_tool.mcp_toolset import MCPToolset,
HttpServerParameters
# Định nghĩa địa chỉ của FastMCP server.
# Đảm bảo fastmcp_server.py của bạn (đã định nghĩa trước đó) đang chạy trên
port này.
FASTMCP_SERVER_URL = "http://localhost:8000"
root_agent = LlmAgent(
model='gemini-2.0-flash', # Hoặc model bạn muốn dùng
name='fastmcp_greeter_agent',
instruction='You are a friendly assistant that can greet people by
their name. Use the "greet" tool.',
tools=[
MCPToolset(
connection_params=HttpServerParameters(
url=FASTMCP_SERVER_URL,
),
# Tùy chọn: Lọc những tool nào từ MCP server được
expose
# Với ví dụ này, ta chỉ kỳ vọng có 'greet'
tool_filter=['greet']
)
],
)
Script này định nghĩa một Agent tên fastmcp_greeter_agent dùng một language model
Gemini. Nó được cung cấp một instruction cụ thể để đóng vai trò trợ lý thân thiện có
mục đích chào mọi người. Quan trọng là code trang bị cho agent này một tool để thực hiện
tác vụ. Nó cấu hình MCPToolset để kết nối tới một server riêng chạy trên
localhost:8000, được kỳ vọng là FastMCP server từ ví dụ
trước. Agent được cấp quyền truy cập cụ thể vào tool greet được host trên
server đó. Về bản chất, code này thiết lập phía client của hệ thống, tạo ra một
agent thông minh hiểu rằng mục tiêu của nó là chào mọi người và biết chính xác
tool bên ngoài nào cần dùng để làm việc đó.
