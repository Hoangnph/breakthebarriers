Agent Card Declaration: Yêu cầu authentication được khai báo rõ trong
Agent Card, một artifact cấu hình mô tả danh tính, capability, và
security policy của agent. Điều này tập trung hóa và đơn giản hóa quản lý authentication.
Credential Handling: Agent thường authenticate bằng credential an toàn như token OAuth
2.0 hoặc API key, được truyền qua HTTP header. Phương pháp này ngăn credential
bị expose trong URL hoặc message body, tăng cường bảo mật tổng thể.
A2A vs. MCP
A2A là một protocol bổ sung cho Model Context Protocol (MCP) của Anthropic (xem
Hình 1). Trong khi MCP tập trung vào cấu trúc context cho agent và tương tác của chúng với
dữ liệu và tool bên ngoài, A2A hỗ trợ điều phối và communication giữa
các agent, cho phép task delegation và collaboration.
