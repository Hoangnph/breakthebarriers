few-shot example prompting, nơi agent học từ các chuỗi tương tác thành công
trước đây để thực hiện tác vụ đúng cách.
Procedural Memory: Ghi nhớ quy tắc: Đây là memory về cách
thực hiện tác vụ - các chỉ dẫn và hành vi cốt lõi của agent, thường nằm trong
system prompt của nó. Việc agent tự sửa prompt của mình để
thích nghi và cải thiện là khá phổ biến. Một kỹ thuật hiệu quả là "Reflection", trong đó agent được
prompt với các chỉ dẫn hiện tại và những tương tác gần đây, rồi được yêu cầu
tinh chỉnh lại chỉ dẫn của chính nó.
Dưới đây là pseudo-code minh họa cách một agent có thể dùng reflection để cập nhật
procedural memory được lưu trong LangGraph BaseStore
# Node cập nhật chỉ dẫn của agent
def update_instructions(state: State, store: BaseStore):
namespace = ("instructions",)
# Lấy các chỉ dẫn hiện tại từ store
current_instructions = store.search(namespace)[0]
# Tạo prompt để yêu cầu LLM phản tư về cuộc hội thoại
# và sinh ra các chỉ dẫn mới, tốt hơn
prompt = prompt_template.format(
instructions=current_instructions.value["instructions"],
conversation=state["messages"]
)
# Lấy các chỉ dẫn mới từ LLM
output = llm.invoke(prompt)
new_instructions = output['new_instructions']
# Lưu các chỉ dẫn đã cập nhật trở lại store
store.put(("agent_instructions",), "agent_a", {"instructions":
new_instructions})
# Node dùng các chỉ dẫn để sinh phản hồi
def call_model(state: State, store: BaseStore):
namespace = ("agent_instructions", )
# Truy xuất các chỉ dẫn mới nhất từ store
instructions = store.get(namespace, key="agent_a")[0]
# Dùng các chỉ dẫn đã truy xuất để định dạng prompt
prompt =
prompt_template.format(instructions=instructions.value["instructions"
