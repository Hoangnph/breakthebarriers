Hình 1: Design pattern quản lý memory
Các điểm chính
Tóm tắt nhanh các ý chính về memory management:
Memory cực kỳ quan trọng để agent theo dõi thông tin, học hỏi, và
cá nhân hóa tương tác.
Conversational AI dựa vào cả short-term memory cho ngữ cảnh tức thời trong
một cuộc chat đơn lẻ và long-term memory cho tri thức bền vững qua nhiều
session.
Short-term memory (phần thông tin tức thời) là tạm thời, thường bị giới hạn bởi
context window của LLM hoặc cách framework truyền ngữ cảnh.
Long-term memory (phần thông tin được giữ lại) lưu thông tin qua các cuộc chat khác nhau
bằng storage bên ngoài như vector database và được truy cập thông qua search.
